# Branding assets

These are **placeholder** images (plain `#070808` with a white "F"). Replace each
with your real Filzy artwork, keeping the **same filename and size** — every path
is already wired into `index.html` and `site.webmanifest`, so nothing else needs
to change.

| File | Size | Used for |
| --- | --- | --- |
| `og-image.png` | 1200×630 | Open Graph / Twitter social preview |
| `favicon-32x32.png` | 32×32 | Browser tab icon |
| `favicon-16x16.png` | 16×16 | Browser tab icon (small) |
| `apple-touch-icon.png` | 180×180 | iOS home screen |
| `android-chrome-192x192.png` | 192×192 | Android / PWA |
| `android-chrome-512x512.png` | 512×512 | Android / PWA (also maskable) |

The vector tab icon lives at [`/public/favicon.svg`](../favicon.svg) — update that too if your mark changes.

Tip: tools like [realfavicongenerator.net](https://realfavicongenerator.net) can
generate this whole set from one source image.
